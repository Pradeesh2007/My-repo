what is dangling images?
  Dangling images in Docker are untagged images

command to list dangling images
  docker images -f "dangling=true"

what is the default location for docker, can cange the default location ,if yes how?
  The default location of docker is "/var/lib/docekr". yes, we can change the docekr default location. copy or move the default docker location(/var/lib/docker) to new path, then edit the file "/etc/docker/daemon.json" and set new docker path you moved.

command to export docker image to sample.tar
  docker save -o sample.tar <image_name>:<tag>

command to find import sample.tar into docker image
  docker load -i sample.tar

diff b/w docker exec vs docker attach
  exec - Runs a new command/process inside an already running container.
  attach - Connects your terminal to the container’s main process.
